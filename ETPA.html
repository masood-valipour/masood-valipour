A comparative study of two-photon state probability distributions in the temporal and frequency domains for three photon-pair architectures: the optimum state, time-energy entangled pairs, and unentangled (classical) pairs. Each section presents interactive visualizations controlled by the physical parameters of the system.

04 · Methods & notes

Computational notes

All visualizations run client-side; no server is required beyond a local file host for the JSON datasets. The JavaScript implementations of the probability functions mirror the analytical expressions derived in the accompanying paper.

Section 1 — Two-photon probability distributions

Calculated on a 200×200 grid using explicit meshgrid evaluation. Complex arithmetic is handled via math.js. The Plotly layout uses a 4-row × 3-column subplot structure. Heatmaps use a reversed Hot colorscale. Grid size and domain can be adjusted via the domain control panel.

  • Optimum state: Analytical expressions in temporal and frequency domains with the two-photon resonance at \(\omega_{fg}\).
  • Entangled state: Gaussian–Gaussian joint spectral amplitude parameterised by pump bandwidth \(\Omega_P\) and correlation bandwidth \(\Omega_M\), with optional delay \(t_{02}\).
  • Unentangled state: Symmetrised product state of two independent Gaussian wave-packets \(\psi_A\) and \(\psi_B\).

Parameters per slider combination are loaded from four pre-computed JSON files in DataSets/.

Section 2 — Peak positions, three states

Roots of the derivative of the marginal distribution \(P(\omega)\) are computed numerically. Each dataset (peak_data_Opt.json, peak_data_Ent.json, peak_data_UnEnt.json) contains pre-calculated peak positions over a grid of \((\Gamma_e/\Gamma_f,\, \delta_a/\Gamma_f)\) values. The slider selects the nearest available \(\delta_a\) row with tolerance 0.5.

Section 3 — Optimum peak positions (full detuning range)

Same root-finding methodology applied to the closed-form expression of the optimum-state marginal (Eq. 33 of the draft). The slider covers \(\delta_a/\Gamma_f \in [0, 100]\) with 0.25 step resolution. Data source: peak_data.json.
