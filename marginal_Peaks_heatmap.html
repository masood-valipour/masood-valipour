Figure Overview and Methodology

This plot displays the marginal distribution p(ω) for the Optimum State as a heatmap background, with the maximum peak positions overlaid on top.

  • Heatmap Background: The color intensity represents the probability density p(ω) computed from the analytical formula for the Optimum State. The distribution shows the probability density across different decay ratios (Γₑ/Γf) and frequencies (ω).
  • Peak Lines (Foreground): The orange and cyan curves trace the positions of local maxima in the distribution, showing how peak positions evolve with the decay ratio.
  • Calculation Method: The marginal distribution is computed using the formula:
    \( p_{\text{Opt}}(\omega) = \frac{\Gamma_e(\Gamma_e + \Gamma_f)(\Gamma_e + \Gamma_f/4) + \Gamma_e(\omega_{fe} - \omega_{eg})^2 + \Gamma_f(\omega - \omega_{eg})^2}{4\pi[(\omega - \omega_{eg})^2 + \Gamma_e^2/4][(\omega - \omega_{fe})^2 + (\Gamma_e + \Gamma_f)^2/4]} \)
    where \(\omega_{eg} = -\delta_a/2\) and \(\omega_{fe} = +\delta_a/2\). The peak positions are found by locating the roots of the derivative of this distribution (Equation 33 of the draft).

Reference Guidelines and Context

Two horizontal reference guidelines provide context for the bidirectional system:

  • The red dashed line corresponds to ωfₑ/Γf = +δₐ/2Γf, representing one of the two-photon resonance frequencies.
  • The blue dashed line corresponds to ωₑg/Γf = -δₐ/2Γf, representing the other resonance frequency.
